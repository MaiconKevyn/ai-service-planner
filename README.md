# AI Service Planner

UI local para estudar e comparar três possibilidades de serviços com IA, Dev e Python no cenário brasileiro:

1. Agente de WhatsApp + mini-CRM para serviços locais.
2. Automação de documentos e backoffice com IA.
3. Copiloto de dados e relatórios em português.

O foco é aprender a transformar capacidade técnica em oferta vendável: objetivo, ideia principal, leads, marketing, custo inicial, stack, riscos e plano de desenvolvimento.

## Rodar localmente

```bash
npm install
npm run dev
```

Depois abra a URL informada pelo Vite, normalmente `http://localhost:5173/`.

## Scripts

```bash
npm run lint
npm run build
npm run verify:ui
npm run preview
```

## Premissas

- A recomendação prioriza serviço produtizado com setup e mensalidade antes de SaaS puro.
- Os custos são estimativas de piloto e variam conforme cliente, volume e provedor.
- Integrações de WhatsApp devem priorizar API oficial ou provedor autorizado.
- Projetos com dados pessoais precisam prever LGPD, retenção, auditoria e handoff humano.
